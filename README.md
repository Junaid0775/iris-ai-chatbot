# 🤖 IRIS AI Chatbot

IRIS AI Chatbot is a modern conversational AI web application powered by a locally hosted Llama 3.2 model using Ollama. It provides a smooth, interactive chat experience with a clean UI and offline functionality.

---

## 🚀 Features

* 🧠 Local AI using Llama 3.2 (Ollama)
* 💬 ChatGPT-like user interface
* 📁 Chat history using localStorage
* ⚡ Fast and fully offline support
* 🔐 Node.js backend for structured API handling
* 🔄 Scalable architecture for future AI integrations

---

## 🏗️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **AI Model:** Llama 3.2 (via Ollama)

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/Junaid0775/iris-ai-chatbot.git
cd iris-ai-chatbot
```

---

### 2️⃣ Install backend dependencies

```
cd backend
npm install
```

---

### 3️⃣ Install Ollama

Download from: https://ollama.com

---

### 4️⃣ Run Ollama

```
ollama serve
```

---

### 5️⃣ Pull Llama model

```
ollama pull llama3.2
```

---

### 6️⃣ Start backend server

```
node server.js
```

---

### 7️⃣ Run frontend

Open the HTML file using Live Server or directly in browser:

```
IRIS_AI_v3_faithful.html
```

---

## 🧪 Usage

1. Start Ollama
2. Start backend server
3. Open frontend
4. Select **Local (Ollama)**
5. Start chatting
